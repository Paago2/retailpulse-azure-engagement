# Sprint Plan

## Sprint 0: Engagement Foundation

**Goal:** Establish client background, business problem, stakeholders, current state, target state, success metrics, and risks.

**Status:** Completed

---

## Sprint 1: Azure DevOps Boards Setup

**Goal:** Create the Azure DevOps project and translate the engagement foundation into Epics, Features, User Stories, and Tasks.

**Deliverables**
- Azure DevOps organization/project created
- Agile process selected
- Initial backlog created
- Epics and Features entered
- User Stories and Tasks entered
- Iterations configured
- Board screenshots captured

---

## Sprint 2: Azure Architecture and Cost-Control Foundation

**Goal:** Design the Azure architecture and prepare a cost-conscious deployment plan.

**Deliverables**
- Architecture diagram
- Azure service selection
- Naming convention
- Resource group plan
- Cost-control checklist
- Budget/alert plan
- Terraform scaffold or portal-based deployment plan

---

## Sprint 3: Batch Data Ingestion

**Goal:** Create sample batch datasets and design the ingestion flow into ADLS Gen2 Bronze.

**Deliverables**
- Sample CSV files
- Source schemas
- Bronze folder structure
- ADF pipeline design
- Validation checklist

---

## Sprint 4: Real-Time Streaming Pipeline

**Goal:** Build the event producer and connect order events to Azure Event Hubs.

**Deliverables**
- Event schema
- Python event producer
- Event Hub setup
- Streaming validation
- Bronze event landing plan

---

## Sprint 5: Data Transformation and Modeling

**Goal:** Build Silver and Gold data models for analytics.

**Deliverables**
- Transformation rules
- Silver tables
- Gold reporting tables
- Data model documentation

---

## Sprint 6: Power BI Reporting and Governance

**Goal:** Define dashboards, KPIs, RLS, data quality rules, and operational handoff.

**Deliverables**
- Dashboard requirements
- KPI glossary
- DAX measure list
- RLS/access plan
- Data quality checklist
- Runbook
