# RetailPulse Azure Data Engagement Platform

## Project Overview

RetailPulse is a simulated Azure data modernization engagement for a mid-sized retail business called Northwind Retail Group.

The goal of the project is to mirror the responsibilities of an Azure Data Engagement & Technical Lead by designing and delivering an end-to-end data platform that supports batch ingestion, real-time event streaming, curated analytics, Power BI reporting, governance, and AI/ML readiness.

## Business Problem

Northwind Retail Group has disconnected data sources across online orders, store transactions, inventory, delivery operations, and customer systems. Leadership lacks trusted, timely reporting and wants a modern Azure-based data platform that improves decision-making and prepares the business for advanced analytics and AI/ML use cases.

## Role Simulated

Azure Data Engagement & Technical Lead

Responsibilities demonstrated:

- Stakeholder discovery
- Current-state and target-state assessment
- Azure data architecture design
- Azure DevOps Boards delivery planning
- Batch and real-time data pipeline design
- BI and reporting strategy
- Data governance and quality planning
- Operational handoff and roadmap development
- AI/ML readiness planning

## Target Azure Architecture

- Azure Data Factory for batch ingestion and orchestration
- Azure Event Hubs for real-time order events
- Azure Data Lake Storage Gen2 for Bronze, Silver, and Gold data layers
- Azure Databricks or Azure Synapse for transformation and analytics
- Power BI for executive and operational dashboards
- Microsoft Purview for governance and cataloging
- Azure DevOps Boards for backlog, sprint, risk, and delivery tracking
- Azure Key Vault and RBAC for security
